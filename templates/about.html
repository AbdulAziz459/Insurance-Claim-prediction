Project Overview
Project Goals
  • Develop accurate insurance claim prediction models
  • Create an intuitive web interface for data analysis
  • Implement comprehensive data preprocessing pipeline
  • Compare multiple machine learning algorithms
  • Provide transparent and explainable predictions
Technologies Used
  • Backend: Python, Flask
  • Machine Learning: Scikit-learn, Pandas, NumPy
  • Visualization: Matplotlib, Seaborn
  • Frontend: HTML5, CSS3, Bootstrap 5, JavaScript
  • Data Processing: SMOTE, StandardScaler

Machine Learning Models

Our application implements and compares seven different machine learning algorithms:

Random Forest

Ensemble method using multiple decision trees

Best for: Feature importance, handling mixed data types
Logistic Regression

Linear model for binary classification

Best for: Interpretability, baseline performance
Support Vector Machine

Kernel-based classification method

Best for: High-dimensional data, non-linear patterns
K-Nearest Neighbors

Instance-based learning algorithm

Best for: Local patterns, non-parametric classification
Decision Tree

Tree-based rule learning method

Best for: Interpretability, handling categorical features
Naive Bayes

Probabilistic classifier based on Bayes' theorem

Best for: Text classification, fast training
Neural Network

Multi-layer perceptron for complex patterns

Best for: Non-linear relationships, large datasets
