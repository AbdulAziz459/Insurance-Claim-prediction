About This Project

This application demonstrates a complete machine learning pipeline for insurance claim prediction using the Porto Seguro dataset. The project includes:

  • Data Analysis: Comprehensive exploration of the dataset structure and characteristics
  • Preprocessing: Advanced data cleaning and preparation techniques
  • Visualization: Interactive charts and graphs for data insights
  • Model Training: Multiple ML algorithms with performance comparison
  • Prediction: Real-time risk assessment for new applications

The goal is to help insurance companies make better decisions about policy pricing and risk assessment while maintaining fairness and transparency.
